Política de Privacidade

Esta política explica como a Almenara Contabilidade trata os dados enviados pelo site para atendimento, contato comercial e continuidade da conversa pelo WhatsApp.

Última atualização: 28/05/2026.

Quem é o responsável pelos dados

A responsável pelo tratamento dos dados é a Almenara Contabilidade LTDA, CNPJ 11.322.275/0001-68, com atendimento em Manhuaçu - MG.

Quais dados podem ser tratados

Quando você preenche o formulário de contato, podemos tratar:

  • nome;
  • empresa;
  • interesse informado no formulário;
  • regime tributário informado;
  • mensagem enviada voluntariamente.

Para que usamos os dados

Usamos os dados para responder ao contato, entender a necessidade informada e iniciar o atendimento contábil solicitado.

O formulário não cria conta, não coleta senha e não armazena os dados em banco próprio do site. Ao enviar, o WhatsApp é aberto com as informações preenchidas para agilizar a conversa.

Não recomendamos o envio de senhas, documentos pessoais, dados bancários ou informações sensíveis pelo campo aberto de mensagem. Caso algum documento seja necessário, a orientação será feita diretamente durante o atendimento.

Compartilhamento com terceiros

Ao escolher enviar pelo WhatsApp, as informações digitadas são encaminhadas para o ambiente do WhatsApp/Meta, conforme a conversa iniciada pelo próprio usuário.

Links externos, como Google Maps, Banco Central, B3, Câmara dos Deputados, Senado Federal e Portal Contábeis, levam o usuário para sites de terceiros, sujeitos às políticas desses respectivos serviços.

Retenção dos dados

As mensagens recebidas pelo WhatsApp podem ser mantidas pelo tempo necessário para atendimento, relacionamento com o cliente, obrigações legais ou defesa de direitos.

Direitos do titular

Você pode solicitar informações sobre seus dados, correção, atualização, eliminação quando aplicável e esclarecimentos sobre o tratamento realizado.

Para exercer seus direitos, entre em contato pelo WhatsApp indicado no site: (33) 8440-1388.

Cookies e monitoramento

O site não utiliza cookies próprios de rastreamento, área de login ou pixels de anúncio. Podemos usar o Vercel Speed Insights para medir desempenho e Core Web Vitals de forma agregada, sem coletar os dados preenchidos no formulário.

A página de notícias usa arquivos públicos locais para exibir notícias e indicadores de mercado.

Atualizações desta política

Esta política pode ser atualizada para refletir mudanças no site, no atendimento ou em exigências legais. A versão vigente será sempre publicada nesta página.
